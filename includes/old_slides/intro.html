Connection to real algebra

In many applications, our goal is to search over a set of functions that satisfy some nonnegativity conditions.
  • Optimization: \(\text{policy}: \text{time} \rightarrow \text{action}\)
  • Duality: $$\min_{x } f(x) \Leftrightarrow \max_{\gamma \in \mathbb R, \;f(x)- \gamma \ge 0 \; \forall x} \gamma$$
  • Analysis of DS $\dot x = f(x)$: \[\text{Stability $\Leftrightarrow$ $\exists$ Lyapunov function $V(x)$}\]
  • Learning: $f: \mathbb R^n \rightarrow \mathbb R^n$
  • $$p(x_1, \ldots, x_n) = \sum_{\alpha_1, \ldots, \alpha_n} p_{\alpha_1, \ldots, \alpha_n} x_1^{\alpha_i} \ldots x_1^{\alpha_n}$$
  • Is $p(x) \ge 0 \quad \forall \in \mathbb R^n$?
  • NP-hard for degrees as small as $4$
  • $\Uparrow$
  • $p(x) = \sum_i q_i^2(x)$
  • Can be cast as an SDP
  • $\iff \exists Q \succeq 0 \quad p(x) = m^TQm$

    with $m = (1, x_1, \ldots, x_n, x_1x_2, \ldots, x_n^d)^T$

  • This is true even if some coefficients of $p$ are unknown
  • Is $p(x) > 0 \quad \forall x \in \mathbb \Omega$?
  • $\Omega = \{x \in \mathbb R^n \; | \; g_1(x) \ge 0, \ldots, g_m(x) \ge 0\}$
  • \[ \; \]
  • Putinar's theorem (under Archimedean condition)
  • $\exists$ sos polynomials $\sigma_0, \ldots, \sigma_m$ s.t. $$p(x) = \sigma_0(x) + \sigma_1(x) g_1(x) + \ldots + \sigma_m(x) g_m(x)$$
  • Search over $\sigma_i$ is an SDP when we bound the degree
